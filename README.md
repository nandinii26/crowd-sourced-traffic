Crowd-Sourced Traffic Prediction System
=====================================

This repository contains a scaffold for a crowd-sourced traffic prediction system.

Contents:
- `architecture/` — PlantUML system & ER diagrams
- `api/` — OpenAPI spec
- `backend/` — Node.js (TypeScript) backend skeleton with Kafka, Redis, PostGIS integration samples
- `mobile/` — Android (Kotlin) mobile app skeleton demonstrating location consent and background collection
- `db/` — SQL schema for PostgreSQL + PostGIS

See each folder's README for details and quick start instructions.
